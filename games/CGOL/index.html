CGOL

CGOL or "Conway's Game Of Life" is a game with only 4 rules, used to simulate life. If you would like to know more, you can click here.

Instructions:
1. Select grid size
2. Generate grid
3. select some some squares to make a pattern
4. Run the game to see if it can survive!
